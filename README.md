## 📱 Sobre o Projeto

<p align="center">
  <img src="Imagens/Icone/organizai_icon.png" alt="OrganizAi" width="200">
</p>

O **OrganizAi** é um aplicativo desenvolvido para ajudar usuários a **organizar suas finanças pessoais de forma simples e intuitiva**, indo além do controle financeiro tradicional.

A aplicação também auxilia o usuário a identificar **benefícios sociais disponíveis**, oferecendo uma experiência personalizada com base no perfil financeiro.

## 💸 Funcionalidades

- ✅ **Perfil Financeiro**  
  Criação de perfil com informações de renda, família e despesas  

- ✅ **Benefícios Personalizados**  
  Sugestão de benefícios governamentais com base no perfil  

- ✅ **Organização Financeira**  
  Controle de gastos, criação de orçamentos e metas
